# Placement Management System Backend

## Overview

The Placement Management System Backend is a server-side application developed to automate and manage campus placement activities.

This system provides APIs for managing students, companies, placement drives, interview schedules, and placement records. The backend is designed to reduce manual work and improve communication between students, placement officers, and recruiters.

---

## Objectives

- Manage student information efficiently
- Maintain company recruitment records
- Track placement status of students
- Schedule interviews and recruitment drives
- Provide secure authentication and authorization

---
